# TFJS Object Detection Demo

## Overview
This project demonstrates real-time object detection in the browser using TensorFlow.js.

## Experiment Results

| Backend | Latency | FPS |
|--------|--------|-----|
| WebGL | ~35 ms | ~28 FPS |
| CPU | ~3000 ms | ~0.3 FPS |

## Conclusion
WebGL provides nearly 100x speedup over CPU, enabling real-time edge AI applications.

## Tech Stack
- TensorFlow.js
- COCO-SSD
- JavaScript
